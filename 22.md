# 第一章　緒論

## 1.1　計畫緣起與研究背景

咖啡為全球消費量最高的飲品之一，近年來隨著連鎖咖啡品牌、超商現煮咖啡與精品咖啡文化在臺灣的普及，咖啡已從少數人的嗜好轉變為多數人日常生活的一部分。對許多上班族、學生與輪班工作者而言，咖啡不僅是提神醒腦的工具，更逐漸成為一種社交與自我獎勵的儀式。然而，咖啡所含之咖啡因屬中樞神經興奮劑，會延長入睡時間、減少深層睡眠並提高夜間覺醒頻率；當飲用量過高或飲用時段過晚時，往往對睡眠品質造成負面影響。

睡眠品質是現代人健康的重要指標。長期睡眠不足或睡眠品質不佳，與情緒障礙、注意力下降、代謝異常及免疫功能降低等問題密切相關。在工作壓力普遍偏高的環境下，許多人一方面倚賴咖啡維持白天的精神狀態，另一方面卻又承受夜晚睡眠不佳的困擾，形成「以咖啡換取清醒、以睡眠品質為代價」的循環。釐清咖啡飲用習慣與睡眠品質之間的關聯，對於個人健康管理具有實務上的參考價值，亦為本研究之核心關切。

基於上述背景，本研究以「咖啡飲用習慣」為自變項、以「睡眠品質」為依變項，透過自編問卷蒐集受訪者之人口統計資料、咖啡飲用行為（是否飲用、每日杯數、飲用種類、選擇因素、購買動機、最後飲用時段、每月花費、生理反應）以及睡眠相關指標（入睡難度、睡眠時數、睡眠滿意度、是否易驚醒），並運用描述性統計、t 檢定、卡方檢定、變異數分析與簡單線性迴歸等方法進行分析，進一步建立決策樹與 K-means 集群模型，期能描繪咖啡族群的睡眠樣貌並提供具體建議。

## 1.2　研究目的

綜合研究動機，本研究之具體目的條列如下：

一、了解受訪者咖啡飲用習慣與睡眠狀況之整體分布，包括飲用比率、每日杯數、飲用時段與睡眠滿意度等描述性特徵。

二、檢定「有無飲用咖啡」之受訪者，在睡眠滿意度與睡眠時數上是否存在顯著差異（t 檢定）。

三、檢定咖啡飲用與否、職業別等類別變項與「睡眠滿意度高低」之關聯（卡方檢定）。

四、比較不同職業別、不同最後飲用時段之受訪者，其睡眠滿意度與入睡難度是否不同（變異數分析與 Tukey 事後比較）。

五、以簡單線性迴歸量化「每日杯數」「入睡難度」「睡眠時數」對睡眠滿意度的影響。

六、建立可預測「高睡眠滿意度」之決策樹模型，並依 TRIPOD 指引進行建立與內部驗證；另以 K-means 集群分析辨識不同睡眠—咖啡型態的族群，提出管理意涵。

本研究之研究架構如圖 1 所示。研究以人口統計變項（性別、年齡、職業、月收入）與咖啡飲用習慣（是否飲用、每日杯數、最後飲用時段等）為前置變項，透過相關分析、t 檢定、卡方檢定與變異數分析探討其與睡眠品質構面（入睡難度、睡眠時數、睡眠滿意度、易驚醒）之關係，最後以迴歸模型與決策樹／集群模型整合分析，形成研究結論與建議。

# 第二章　研究方法

## 2.1　研究對象與資料來源

本研究以一般民眾為研究對象，透過網路問卷（Google 表單）進行匿名自填式調查，蒐集受訪者之人口統計資料、咖啡飲用習慣與睡眠品質相關資訊。問卷初步回收有效樣本 43 份。由於樣本數明顯不足，難以支撐 t 檢定、變異數分析與迴歸等推論統計所需之檢定力，且本課程作業要求資料筆數至少達 300 筆，故本研究在保留原始 43 筆真實樣本的前提下，另以「同分配隨機生成（same-distribution simulation）」方式擴增資料，詳見第 2.3 節。擴增後之總樣本數為 320 筆（原始 43 筆、模擬 277 筆），其中有飲用咖啡者 190 人、無飲用者 130 人。

須特別說明的是，原始樣本數偏低為本研究之先天限制；模擬資料雖係依原始問卷之邊際分配與合理之關聯結構生成，能維持各變項之分布特性並提供足夠之分析筆數，但其本質仍為模擬，無法完全等同於真實母體。後續所有統計推論之解讀，皆應在此前提下審慎看待，相關限制亦於第六章再次說明。

## 2.3　資料建檔與處理

資料處理流程包含「原始資料清理」「同分配隨機生成擴增」「遺失值注入」與「衍生變數建立」四個步驟，茲分述如下。

一、原始資料清理：原始問卷之「每日杯數」題項回答格式不一，部分受訪者以毫升（ml）作答。本研究將明顯以毫升填答者（數值大於 10）以每杯約 240 毫升換算為杯數；睡眠時數超過 12 小時者視為填答錯誤，以遺失值處理；月收入欄位中「不便透露」則編碼為遺失值。

二、同分配隨機生成擴增：以原始 43 筆樣本之各變項邊際分配為基礎，並設定一潛在「睡眠體質」因子，使「咖啡飲用量增加、飲用時段越晚、輪班工作」等情境對應較差之睡眠結果，據此模擬產生 277 筆資料，與原始樣本合併為 N＝320。此步驟同時提供 SAS 版本之資料生成程式（附錄，程式 01），以 RAND 系列亂數函數示範相同之生成邏輯。

三、遺失值注入：為符合資料須含「至少 10 個（排除跳答）遺失值」之要求，於非跳答欄位（年齡、睡眠時數、睡眠滿意度、易驚醒、每月花費）以完全隨機（MCAR）方式注入遺失值，合計 26 個。連同月收入之「不便透露」與題項清理所產生之遺失，整體資料之遺失情形更貼近真實調查。

## 2.4　統計分析方法

本研究之資料分析以 SAS 進行（隨附程式 02 可完整重現報告數據）。分析流程如圖 2，依序為描述性統計、推論統計與預測／分群模型，茲將各方法說明如下。

![](data:image/png;base64...)

圖 2　資料分析流程圖

一、描述性統計：以次數、百分比、平均數、標準差、中位數與四分位數，呈現各變項之分布情形（PROC FREQ、PROC MEANS、PROC UNIVARIATE）。

二、t 檢定：比較「有無飲用咖啡」及「性別」兩組在睡眠滿意度、睡眠時數上的平均差異（PROC TTEST）。

三、卡方檢定：檢定類別變項（飲用與否、職業、性別、最後飲用時段）與「睡眠滿意度高低／是否易驚醒」之關聯（PROC FREQ 之 CHISQ 選項），並以 Cramér’s V 衡量關聯強度。

四、變異數分析（ANOVA）：比較不同職業別之睡眠滿意度、不同最後飲用時段之入睡難度，並以 Tukey HSD 進行事後多重比較（PROC GLM）。

五、簡單線性迴歸：量化每日杯數、入睡難度、睡眠時數對睡眠滿意度之線性影響（PROC REG）。

六、決策樹：以 PROC HPSPLIT 建立可預測「高睡眠滿意度」之分類樹；依 TRIPOD（個體預後或診斷之多變項預測模型透明化報告）指引，採 70% 建立、30% 內部驗證之資料切分，並報告分類正確率與 AUC。

七、K-means 集群分析：以 PROC FASTCLUS 將受訪者依年齡、睡眠時數、睡眠滿意度、易驚醒與每日杯數分群，辨識不同之睡眠—咖啡型態族群。

所有顯著性檢定之顯著水準均設為 α＝0.05。為使數據可完全重現，正式分析以隨附之coffee\_sleep.csv 為單一資料來源，由程式 02 匯入後執行上述所有分析；本報告中之圖表數據即由該資料計算而得。

# 第四章　推論統計

本章針對研究假設，依序進行 t 檢定、卡方檢定、變異數分析與簡單線性迴歸，顯著水準均設為 α＝0.05。

## 4.1　t 檢定

為檢定「有無飲用咖啡」兩組在睡眠品質上之差異，分別以睡眠滿意度與睡眠時數為依變項進行獨立樣本 t 檢定（圖 15、表 13）。結果顯示，有飲用咖啡者之睡眠滿意度平均 57.68 分，顯著低於無飲用者之 69.07 分（t＝-6.029，p<.001）；睡眠時數方面，飲用者平均 6.74 小時亦顯著少於無飲用者之 7.23 小時（t＝-4.156，p<.001）。兩項結果均支持「飲用咖啡與較差睡眠品質相關」之假設。

# 第五章　預測與分群模型

為超越單變項之關聯檢定，本章進一步建立兩類模型：以決策樹預測個體是否具有高睡眠滿意度，並以 K-means 集群分析辨識不同之睡眠—咖啡型態族群。

## 5.1　決策樹模型（依 TRIPOD 指引）

本研究依 TRIPOD（個體預後或診斷之多變項預測模型透明化報告）指引建立並報告決策樹模型。結果變數為「睡眠滿意度是否達高滿意（≧70 分）」，預測因子包含年齡、月收入級距、每日杯數、最後飲用時段、入睡難度、每月花費及性別、輪班、是否飲用等。為避免與結果重複定義之變項造成的循環論證，刻意排除「易驚醒」此一同屬睡眠結果之變項。資料以 7：3 切分為建立組（n＝219）與內部驗證組（n＝95），樹深限制為 3、葉節點最小樣本數 15。

模型於驗證組之分類正確率為 0.684、AUC 為 0.709（圖 21、表 21），具備可接受之區辨能力。由根節點起首要分割變項為入睡難度，其次為年齡與收入級距；變數重要性（圖 22、表 22）顯示年齡（0.481）與入睡難度（0.451）為最關鍵之兩項預測因子，咖啡相關行為變項之邊際重要性相對較低。此結果提示：在控制睡眠主觀困難與年齡後，單純之咖啡飲用量對「是否高滿意」之增額預測力有限，咖啡的影響較可能透過入睡難度等中介路徑展現。

## 5.3　模型之管理意涵

綜合決策樹與集群分析結果，可得三項管理與健康促進意涵。其一，入睡難度與年齡為辨識睡眠品質之關鍵指標，健康衛教可優先針對主觀入睡困難者進行篩檢與介入。其二，「重度咖啡‧受擾群」具有飲用量高、飲用偏晚與睡眠困擾並存之特徵，宜作為衛教與產品設計（如低咖啡因、午後限定品項）之重點對象。其三，咖啡之影響較可能透過入睡難度等中介歷程作用，故減量與調整飲用時段之建議，較單純戒斷更具可行性與接受度。

# 第六章　結論與建議

## 6.1　研究發現

本研究以 320 筆資料分析咖啡飲用習慣與睡眠品質之關聯，主要發現如下：

一、有飲用咖啡者之睡眠滿意度與睡眠時數均顯著低於無飲用者，且飲用者落入「低睡眠滿意度」之比例顯著較高（t 檢定與卡方檢定皆達顯著）。

二、職業別與睡眠品質關係密切，輪班制受訪者之睡眠滿意度顯著最低；最後飲用時段越晚者，入睡難度顯著越高（ANOVA 與 Tukey 事後比較）。

三、迴歸分析顯示每日杯數、入睡難度與睡眠滿意度呈顯著負向關係，睡眠時數則與滿意度呈顯著正向關係，其中入睡難度與睡眠時數之解釋力最強。

四、決策樹（AUC＝0.709）指出入睡難度與年齡為最重要之預測因子；K-means 則辨識出「重度咖啡‧受擾群」此一高風險族群。

## 6.2　研究限制

本研究有以下限制應予說明。第一，原始有效樣本僅 43 份，為達分析所需筆數，係以同分配隨機生成擴增至 320 筆；模擬資料雖保留原始分布特性，惟其內含之關聯結構部分出於研究者之合理假設，故顯著性結果之外推應審慎，不宜逕行推論至真實母體。第二，研究為橫斷面設計，僅能呈現變項間之關聯，無法確立因果方向。第三，咖啡飲用量與睡眠指標多為自陳資料，可能存在回憶與社會期許偏誤。第四，決策樹之區辨能力屬中等，且僅完成內部驗證，未來宜以獨立樣本進行外部驗證。

## 6.3　建議

基於研究發現，提出下列建議。對個人而言，建議將咖啡飲用集中於上午至午後早段，避免於 18 時後飲用，並留意每日總量，以降低對入睡與睡眠時數之干擾。對輪班工作者等高風險族群，建議搭配規律作息與睡眠衛生教育，必要時尋求專業協助。對後續研究而言，建議擴大真實樣本、採用客觀睡眠量測（如穿戴裝置）並進行縱貫追蹤，以強化因果推論與模型之外部效度。

**程式 01：資料生成（01\_資料生成.sas）**

/\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*

進階 SAS 程式設計 期末報告

程式 01：以 SAS 亂數函數「同分配隨機生成」咖啡飲用與睡眠資料

說明：本程式依原始問卷 (n=43) 之邊際分配與假設之關聯結構，

模擬產生 277 筆資料，與原始樣本合併為 N=320。

※ 為使報告數據可完全重現，正式分析請以隨附之

coffee\_sleep.csv（程式 02 匯入）為準；本程式示範

題目所要求之「以 SAS 亂數函數產生資料」之方法。

\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*/

options nodate nonumber;

%let SEED = 20260617;

data sim;

call streaminit(&SEED);

length gender $2 occ $12;

do id = 44 to 320; /\* 與原始 43 筆銜接 \*/

/\* 性別：男 0.55 \*/

gender = ifc(rand("uniform") < 0.55, "男", "女");

/\* 年齡：右偏（gamma），集中於 20-30，截尾 18-72 \*/

age = round(20 + rand("gamma", 2.0) \* 4.2);

if age < 18 then age = 18; if age > 72 then age = 72;

/\* 職業：依年齡條件 \*/

u = rand("uniform");

if age >= 62 then do;

if u<0.70 then occ="退休"; else if u<0.90 then occ="一般上班族"; else occ="自由接案者";

end;

else if age <= 23 then do;

if u<0.78 then occ="學生"; else if u<0.93 then occ="一般上班族"; else occ="輪班制";

end;

else do;

if u<0.50 then occ="一般上班族"; else if u<0.68 then occ="學生";

else if u<0.86 then occ="自由接案者"; else if u<0.98 then occ="輪班制"; else occ="退休";

end;

/\* 月收入級距 1-5（.=不便透露/未填） \*/

v = rand("uniform");

if occ="學生" then income = ifn(v<0.70,1, ifn(v<0.85,2,.));

else if occ="退休" then income = ifn(v<0.40,1, ifn(v<0.70,2, ifn(v<0.85,3,.)));

else do;

if v<0.18 then income=1; else if v<0.52 then income=2; else if v<0.74 then income=3;

else if v<0.86 then income=4; else if v<0.92 then income=5; else income=.;

end;

/\* 潛在睡眠體質 Z（越大越好），輪班/高齡較差 \*/

shiftpen = ifn(occ="輪班制", -0.7, 0);

Z = rand("normal") + shiftpen - 0.012\*(age-28);

/\* 是否飲用咖啡 0.62 \*/

drink = (rand("uniform") < 0.62);

cups=.; lasttime=.; sleepdiff=.; spend=.;

if drink=1 then do;

cups = 0.5 + rand("gamma", 2.0)\*0.7;

cups = round(cups\*2)/2; if cups<0.5 then cups=0.5; if cups>5 then cups=5;

/\* 最後飲用時段 1-4，杯數多者偏晚 \*/

lt = rand("uniform");

if lt<0.18 then lasttime=1; else if lt<0.50 then lasttime=2;

else if lt<0.80 then lasttime=3; else lasttime=4;

sleepdiff = round(1 + 0.55\*cups + 0.45\*(lasttime-1) + rand("normal")\*0.8);

if sleepdiff<1 then sleepdiff=1; if sleepdiff>5 then sleepdiff=5;

spend = 1 + (cups>1.5) + (cups>3) + (rand("uniform")<0.15);

if spend>3 then spend=3;

end;

/\* 睡眠結果（全體） \*/

cups\_eff = coalesce(cups,0);

late\_eff = coalesce(lasttime,1)-1;

sleephr = round(7.3 - 0.18\*cups\_eff - 0.18\*late\_eff + 0.55\*Z + rand("normal")\*0.7, 0.1);

if sleephr<3.5 then sleephr=3.5; if sleephr>11 then sleephr=11;

sd\_eff = ifn(missing(sleepdiff),0,sleepdiff-1);

drinkpen = ifn(drink=1, -3, 0);

sleepsat = round(70 + 11\*Z + 3.5\*(sleephr-7) - 4.2\*sd\_eff - 1.6\*cups\_eff + drinkpen + rand("normal")\*7);

if sleepsat<0 then sleepsat=0; if sleepsat>100 then sleepsat=100;

wake = round(3 - 1.15\*Z + 0.18\*cups\_eff + 0.12\*late\_eff + rand("normal")\*0.7);

if wake<1 then wake=1; if wake>5 then wake=5;

output;

end;

drop u v lt shiftpen Z cups\_eff late\_eff sd\_eff drinkpen;

run;

/\* 隨機注入非跳答遺失值（MCAR），確保 >=10 個 \*/

data sim\_miss; set sim;

call streaminit(&SEED + 7);

if rand("uniform")<0.02 then age=.;

if rand("uniform")<0.02 then sleepsat=.;

if rand("uniform")<0.015 then sleephr=.;

if rand("uniform")<0.012 then wake=.;

if drink=1 and rand("uniform")<0.02 then spend=.;

run;

proc means data=sim\_miss n nmiss mean std min max maxdec=2;

var age cups sleephr sleepsat wake;

run;

title "模擬資料之飲用習慣分配"; proc freq data=sim\_miss; tables drink gender occ; run; title;

==================================================================\*/

title "ANOVA：不同職業 之 睡眠滿意度差異";

proc glm data=coffee;

class occ;

model sleepsat = occ;

means occ / tukey hovtest;

run; quit;

title "ANOVA：不同最後飲用時段 之 入睡難度差異（飲用者）";

proc glm data=coffee;

where drink=1;

class last\_lab;

model sleepdiff = last\_lab;

means last\_lab / tukey;

run; quit;

title;

/\*==================================================================

六、簡單線性迴歸

==================================================================\*/

title "簡單線性迴歸：每日杯數 -> 睡眠滿意度（飲用者）";

proc reg data=coffee;

where drink=1;

model sleepsat = cups / clb;

run; quit;

title "簡單線性迴歸：入睡難度 -> 睡眠滿意度（飲用者）";

proc reg data=coffee;

where drink=1;

model sleepsat = sleepdiff / clb;

run; quit;

title "簡單線性迴歸：睡眠時數 -> 睡眠滿意度（全體）";

proc reg data=coffee;

model sleepsat = sleephr / clb;

run; quit;

title;

/\*==================================================================

七、決策樹（Decision Tree, PROC HPSPLIT）

結局指標：睡眠滿意度是否達高滿意（sat\_hi = sleepsat>=70）

依 TRIPOD：建模(70%)＋內部驗證(30%)

==================================================================\*/

/\* 連續預測因子之缺失以中位數插補（單一插補） \*/

proc stdize data=coffee out=coffee\_imp method=median reponly;

var age income;

run;

data coffee\_imp;

set coffee\_imp;

/\* 飲用者專屬變數對非飲用者以 0 帶入，使全體可進入模型 \*/

cups0 = coalesce(cups, 0);

lasttime0 = coalesce(lasttime, 0);

sleepdiff0= coalesce(sleepdiff, 0);

spend0 = coalesce(spend, 0);

if not missing(sleepsat) then sat\_hi = (sleepsat >= 70);

run;

title "決策樹：預測高睡眠滿意度（TRIPOD 建立＋內部驗證）";

proc hpsplit data=coffee\_imp maxdepth=3 minleafsize=15 seed=42;

class sat\_hi male shift drink;

model sat\_hi(event='1') = age income cups0 lasttime0 sleepdiff0

spend0 male shift drink;

partition fraction(validate=0.3 seed=42);

prune costcomplexity;

run;

title;

/\*==================================================================

八、K-means 集群分析（PROC FASTCLUS）

分群變數：年齡、睡眠時數、睡眠滿意度、易驚醒、每日杯數(0 帶入)

==================================================================\*/

/\* 先標準化（平均 0、標準差 1），避免量綱影響距離 \*/

proc stdize data=coffee\_imp out=coffee\_std method=std;

var age sleephr sleepsat wake cups0;

run;

title "K-means 集群分析（k=3）";

proc fastclus data=coffee\_std out=clusout maxclusters=3 maxiter=50 seed=42;

var age sleephr sleepsat wake cups0;

run;

/\* 將分群結果還原至原始量尺檢視各群輪廓 \*/

data clus\_join;

merge coffee\_imp clusout(keep=cluster);

run;

title "各集群之變數平均（原始量尺）";

proc means data=clus\_join mean maxdec=2;

class cluster;

var age sleephr sleepsat wake cups0 drink;

run;

title;

/\* 全部完成 \*/
