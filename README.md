# 咖啡飲用習慣與睡眠品質關聯性研究

> 淡江大學　114 學年度第 2 學期　進階 SAS 程式設計　期末報告

以自編問卷調查咖啡飲用習慣與睡眠品質之關聯，並以 **SAS** 完成從資料生成、描述統計、推論統計，到決策樹與 K-means 集群之完整分析流程。

---

## 📌 研究摘要

- **自變項：** 咖啡飲用習慣（是否飲用、每日杯數、飲用種類、選擇因素、購買動機、最後飲用時段、每月花費、生理反應）
- **依變項：** 睡眠品質（入睡難度、睡眠時數、睡眠滿意度、是否易驚醒）
- **樣本：** 320 筆（原始問卷 43 筆 ＋ 同分配隨機生成 277 筆），其中飲用者 190 人、非飲用者 130 人
- **工具：** SAS（PROC FREQ / MEANS / TTEST / GLM / REG / HPSPLIT / FASTCLUS）
- **顯著水準：** α = 0.05

### 主要發現

| 主題 | 結果 |
| --- | --- |
| t 檢定 | 飲用者之睡眠滿意度（57.7）與睡眠時數（6.74 hr）均**顯著低於**非飲用者（69.1 分、7.23 hr），p < .001 |
| 卡方檢定 | 飲用咖啡與「低睡眠滿意度」顯著相關（χ²=17.42, p<.001）；輪班族落入低滿意比例最高 |
| ANOVA | 輪班族睡眠滿意度顯著最低；飲用時段越晚（18–21 時）入睡難度越高 |
| 簡單線性迴歸 | 每多 1 杯咖啡，滿意度↓4.01 分；入睡難度與睡眠時數對滿意度解釋力最強 |
| 決策樹（AUC=0.709） | 入睡難度與年齡為最關鍵預測因子，咖啡量的影響多透過入睡難度中介 |
| K-means（k=3） | 辨識出「重度咖啡‧受擾群」此一高風險族群（睡眠最短、滿意度最低、最易驚醒） |

## 🗂️ 資料檔說明（coffee_sleep.csv）

共 **320 筆、39 個欄位**，欄位以英文命名、值以中文呈現。`src` 欄標記「原始／模擬」。

| 欄位群組 | 欄位 | 說明 |
| --- | --- | --- |
| 人口統計 | `gender` `age` `occ` `income` | 性別／年齡／職業／月收入級距(1–5) |
| 飲用行為 | `drink` `cups` `lasttime` `spend` | 是否飲用(1/0)／每日杯數／最後飲用時段(1–4)／每月花費 |
| 睡眠變項 | `sleepdiff` `sleephr` `sleepsat` `wake` | 入睡難度(1–5)／睡眠時數／睡眠滿意度(0–100)／易驚醒(1–5) |
| 咖啡種類（複選） | `t_*` | `t_americano` `t_latte` `t_cappu` `t_mocha` `t_other` |
| 選擇因素（複選） | `f_*` | `f_taste` `f_price` `f_conv` `f_origin` `f_caf` `f_brand` `f_flavor` |
| 生理反應（複選） | `r_*` | `r_none` `r_freq` `r_excite` `r_palpit` `r_anx` `r_tremor` `r_appet` |
| 購買動機（排序） | `m_*` | `m_refresh` `m_ritual` `m_physio` `m_flavor` `m_reward` `m_social` |
| 來源標記 | `src` | 原始／模擬 |

- 複選題以多個 0／1 旗標欄位（`t_` / `f_` / `r_` 前綴）表示。
- 排序題以 `m_` 前綴欄位記錄順位。
- 含遺失值（月收入「不便透露」、MCAR 注入等），更貼近真實調查。

## ▶️ 如何重現分析

需要 **SAS**（SAS Studio / OnDemand for Academics / PC-SAS 皆可）。

1. 將 `coffee_sleep.csv` 放到 SAS 可存取的路徑。
2. 開啟 `02_資料匯入與分析.sas`，修改開頭的路徑巨集為你的實際檔案位置：
   ```sas
   %let PATH = /home/u_data/coffee_sleep.csv;   /* 改成你的路徑 */
   ```
3. 由於檔案含中文字元，請確認 SAS 工作階段為 **UTF-8** 編碼
   （SAS Studio / OnDemand 預設即為 UTF-8；本機 PC-SAS 若為 BIG5，請以 UTF-8 重新開啟，或在啟動參數加上 `-encoding utf-8`）。
4. 執行程式，即可重現報告中所有描述統計、t 檢定、卡方、ANOVA、迴歸、決策樹與 K-means 結果。

> `01_資料生成.sas` 可獨立執行，示範同分配隨機生成的邏輯（固定亂數種子 `SEED=20260617`）。

## 🔬 分析方法總覽

| 章節 | 方法 | SAS 程序 |
| --- | --- | --- |
| 描述統計 | 次數、百分比、平均、標準差、四分位數 | `PROC FREQ` / `MEANS` / `UNIVARIATE` |
| t 檢定 | 飲用與否、性別 對 睡眠變項 | `PROC TTEST` |
| 卡方檢定 | 類別變項與睡眠滿意度高低之關聯（含 Cramér's V） | `PROC FREQ (CHISQ)` |
| 變異數分析 | 職業別、飲用時段差異（Tukey HSD 事後比較） | `PROC GLM` |
| 簡單線性迴歸 | 杯數／入睡難度／睡眠時數 → 睡眠滿意度 | `PROC REG` |
| 決策樹 | 預測高睡眠滿意度，依 TRIPOD 指引 70/30 切分驗證 | `PROC HPSPLIT` |
| K-means 集群 | 依睡眠—咖啡型態分群 | `PROC FASTCLUS` |

## ⚠️ 研究限制

- 原始有效樣本僅 43 份，係以同分配隨機生成擴增至 320 筆；模擬資料雖保留原始分布特性，惟其關聯結構部分出於合理假設，**顯著性結果不宜逕行外推至真實母體**。
- 橫斷面設計，僅能呈現關聯，無法確立因果方向。
- 咖啡飲用與睡眠指標多為自陳資料，可能存在回憶與社會期許偏誤。
- 決策樹僅完成內部驗證，未來宜以獨立樣本進行外部驗證。
